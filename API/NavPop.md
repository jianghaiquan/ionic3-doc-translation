# NavPop
`[navPop]`  
这是一个声明式指令，用于退出当前页面。
## 使用

```
<ion-content>

 <button ion-button navPop>Go Back</button>

</ion-content>
```

类似的还有[NavPush](http://ionicframework.com/docs/api/components/nav/NavPush/)
